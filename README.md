# Python Basics

This folder contains introductory Python examples for core language concepts. Each file demonstrates a different fundamental topic, making this a good reference for learning or reviewing Python syntax and behavior.

## Contents

- `built_in_functions.py` - Examples of Python built-in functions such as `print()`, `len()`, `type()`, and others.
- `comments.py` - Demonstrates how to write comments in Python using `#` and multi-line string comments.
- `comparison_operators.py` - Shows comparison operators like `==`, `!=`, `<`, `>`, `<=`, and `>=`.
- `conditional_statements.py` - Covers conditional logic using `if`, `elif`, and `else`.
- `data_structures.py` - Introduces Python collections such as lists, tuples, sets, and dictionaries.
- `data_types.py` - Demonstrates basic Python data types including strings, integers, floats, booleans, and type conversion.
- `for_loops.py` - Examples of `for` loops, iterating over sequences and using `range()`.
- `logical_operators.py` - Shows logical operators like `and`, `or`, and `not`.
- `memberships.py` - Demonstrates membership tests using `in` and `not in`.
- `variables.py` - Covers variable assignment, naming, and simple operations.
- `while_loops.py` - Examples of `while` loops and loop control.

## Usage

Run any example file with Python from this folder:

```bash
python <filename>.py
```

For example:

```bash
python variables.py
```

## Notes

- These examples are designed for beginners.
- Modify the files to explore how each feature behaves.
